DIV Y SPAN

El elemento div nos permite crear secciones en una página web Los únicos elementos que se utilizan para crear secciones en una página web son el DIV y SPAN. El elemento DIV es de los llamados a nivel bloque y el SPAN a nivel de línea; el div permite crear capas a nivel de bloque y el SPAN a nivel de línea. Lo cual significa que a un elemento DIV el navegador para representarlo añade un salto de línea al principio y al final del documento. Los elementos que son a nivel de línea SPAN no llevan salto de línea ni al principio ni al final. El elemento DIV puede agrupar a cualquier conjunto de elemento fundamentalmente de lo llamado a nivel de bloques, es decir que encierra elementos a nivel de bloque, es un CONTENEDOR al igual que el SPAN, a ambos también se les llama contenedores. Visualmente no aportan nada si no incluyes elementos en su interior, ni el DIV y el SPAN. Además el elemento DIV permite solapar contenido, permite que los elementos que se encierran en el puedan estar en distintos niveles de profundidad. Y además permite ocultar o mostrar estos elementos, con la propiedad VISIBILITY.

La posición de cualquier elemento puede ser estática, relativa, absoluta y fija. Y cada uno de estos valores tendrá un significado diferente. Por defecto el valor es estático, que quiere decir que los elementos los representa el navegador de izquierda a derecha y de arriba abajo.
