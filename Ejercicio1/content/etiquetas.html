Ejercicio Actividad 2 - Parte 2

Mostrar una lista no ordenada de cinco de las etiquetas html que hemos estudiado con sus correspondientes definiciones.

<head>
El elemento head define la cabecera del documento y contiene información del mismo (metadatos, scripts, estilos, ubicación de documentos de estilos, título de la página, etc.).
<base>
La etiqueta base se utiliza para definir una dirección por defecto para todos los enlaces de la página. Esta etiqueta debe ir ubicada en la sección head del documento.
<strong>
La etiqueta strong se usa para indicar énfasis fuerte. Los navegadores presentan el texto generalmente con una fuente en negrita.
<em>
La etiqueta em se usa para indicar énfasis. Los navegadores presentan el texto enfatizado generalmente en itálica.
<span>
La etiqueta span se utiliza para añadir estructura a una parte de un texto. Esta etiqueta se usa comunmente para añadir un estilo determinado dentro de una sección del texto.
